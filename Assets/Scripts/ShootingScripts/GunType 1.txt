how to build this scripts:

-Under player gameobjet insert a new game object
	that must have shoot system
-Selector set different type
	of shoot system:
		o: simple(in oone frame)
		1: complex(speard in multiple frame)

-Gun insert Gun script for the gun value
-Muzzle insert Muzzle script for the 
	position of muzzle


Different shooting type:
num 0: single shoot
num 1: multi shoot
